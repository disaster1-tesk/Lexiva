# 2026-04-11 工作日志

## 修复内容

### 问题1: AI对话没有走配置的模型能力
- 原因：chat_service 使用 _settings_cache 缓存，保存设置后不生效
- 修复：
  - backend/services/ai_chat.py 添加 reload_settings() 方法清除缓存
  - backend/api/settings.py 保存设置后调用 chat_service.reload_settings()

### 问题2: 发音评测点击开始没有响应
- 原因：后端缺少 /api/pronunciation/sentence/{id}/audio 端点
- 修复：backend/api/pronunciation.py 添加 TTS 音频端点

### 问题3: 听力训练生成的音频无法播放
- 原因：后端返回 base64 编码，前端直接当二进制处理
- 修复：
  - frontend/src/api/index.ts: 移除 responseType: 'blob'
  - frontend/src/views/Listening.vue: 使用 atob() 解码 base64
  - frontend/src/views/Vocabulary.vue: 同样修复 playPronunciation

### 问题4: 写作批改点击提交没有生效
- 原因：后端返回字段与前端期望不匹配
- 修复：frontend/src/views/Writing.vue submitWriting 添加数据转换逻辑
